# 小学数学练习题生成器

这是一个用于生成小学数学练习题的网页应用程序，支持多种题型并可将生成的题目导出为PDF格式。

## 功能特点

- **多种题型支持**：
  - 加法运算
  - 减法运算
  - 加减混合运算
  - 乘法运算（九九乘法表）
  - 除法运算
  - 四则混合运算
  - 带括号的加减运算
  - 带括号的乘除运算
  - 带小括号的四则混合运算

- **题目类型选择**：
  - 下拉菜单选择题目类型，支持多选
  - 可自由组合不同类型的题目
  - 默认全选所有题目类型

- **题目生成**：
  - 自动生成100道数学练习题
  - 每道题都保证有合理的答案范围
  - 题目以网格形式展示，便于打印

- **导出功能**：
  - 一键打印功能
  - 导出为PDF格式，方便保存和分享

## 严格限制条件

为确保题目适合小学生练习，系统实施了以下严格限制：

1. **结果范围限制**：所有算式的结果都在0到100的范围内
2. **减法限制**：任何减法运算都不会出现负数结果
3. **加法限制**：任何加法运算的结果都不会超过100
4. **乘法限制**：任何乘法运算都不会出现数字1
5. **除法限制**：任何除法运算都不会出现数字0作为除数，也不会出现数字1

## 使用方法

1. 打开 `index.html` 文件
2. 通过下拉菜单选择需要的题目类型（默认全选）
3. 点击"生成题目"按钮创建新的练习题
4. 使用"打印"按钮直接打印练习题
5. 使用"导出为PDF"按钮将练习题保存为PDF文件

## 技术实现

- 纯前端实现，无需服务器支持
- 使用HTML、CSS和JavaScript
- 集成html2pdf.js库实现PDF导出功能
- 响应式设计，适配打印格式

## 题目类型说明

- **加法题目**：生成两个数相加的题目，和不超过100，不包含数字1
- **减法题目**：生成减法题目，确保差值不小于0，不包含数字1
- **乘法题目**：基于九九乘法表的乘法运算，不包含数字1
- **除法题目**：基于九九乘法表的除法运算，确保整除，不包含数字0和1
- **加减混合**：包含两个运算符的混合运算，结果在0-100之间，不包含数字1
- **混合运算**：加减乘除混合的四则运算，结果在0-100之间，不包含数字1，除法不包含数字0。**重要：混合运算的每一步都必须遵守上述基本规则，例如第一步的结果也必须在0-100之间，乘法和除法也必须来自九九乘法表**

- **带括号的加减运算**：只包含加减法和括号的运算，有两种形式：
  - (a ± b) ± c
  - a ± (b ± c)
  结果在0-100之间，不包含数字1。**重要：带括号运算的每一步都必须遵守上述基本规则，括号内的运算结果也必须在0-100之间**

- **带括号的乘除运算**：只包含乘除法和括号的运算，基于九九乘法表，有两种形式：
  - (a × b) ÷ c 或 (a ÷ b) × c
  - a × (b ÷ c) 或 a ÷ (b × c)
  结果在0-100之间，不包含数字0和1，除法确保能整除。**重要：带括号运算的每一步都必须遵守上述基本规则，括号内的运算结果也必须在0-100之间**

- **带小括号的四则混合运算**：包含加减乘除和括号的混合运算，但有特殊限制：
  - 括号外只能是乘法或除法
  - 括号内只能是加法或减法
  - 有两种形式：
    - (a ± b) × c 或 (a ± b) ÷ c
    - a × (b ± c) 或 a ÷ (b ± c)
  结果在0-100之间，不包含数字1，除法不包含数字0且确保能整除。**重要：带括号运算的每一步都必须遵守上述基本规则，括号内的运算结果也必须在0-100之间**

## 注意事项

- 所有题目均保证答案在合理范围内（0-100）
- 除法题目确保能整除，不会出现小数
- 生成的PDF文件会自动包含生成日期和所选题目类型
- 所有混合运算和带括号运算的每一步都遵守基本运算规则
